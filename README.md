# Cloud9: An Airline Booking Web App

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

This branch uses both Paystack and Stripe to receive payments. Paystack for ` M-Pesa ` and Stripe for card payments like ` VISA ` and ` Mastercard `, ` Google PAY ` and ` Apple Pay `. The Stripe account is in ` test mode `. Stripe has yet to allow users from my country to have live accounts
