# Cloud9: An Airline Booking Web App

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

This branch uses only Paystack for receiving payments; both for ` M-Pesa ` and card payments, ` VISA `, ` Mastercard ` and ` American Express(AMEX) `
